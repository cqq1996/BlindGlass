  • 项目描述

    基于树莓派，AI算法，java后台，阿里云、百度云相关服务，uni-app实现的可以辅助盲人出行的智能语音播报机器人

  • 实现功能
    1. 实时定位
    2. 街道目标与盲道图像及时识别
    3. 查询附近地点
    4. 监护人与盲人及时通讯(文字语音相互转换)
    5. 说出目的地唤醒百度地图语音导航

  • 环境准备
    主体 环境or服务
    树莓派 内存>8G TF卡，500w像素摄像头，USB免驱声卡，3.5mm音响或耳机，3.5mm麦克风,USB锂电池，Python3.5
    Java后台 JDK8,Tomcat9,Maven3.6.0,Spring5.1.1,Ubuntu16.0.4
    uni-app HBuilderX,Node.js,Vue.js
    AI算法 Ubuntu16.0.4,Python2.7,Tensorflow1.6,anaconda1.6,conda4.3
    阿里云 ECS服务器,对象存储OSS,备案域名,视频直播,CDN域名加速
    百度API 百度在线语音识别Python SDK,百度地图web API
